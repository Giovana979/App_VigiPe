# 🇧🇷 VigiPeçonha - Aplicativo de Vigilância Urbana
O VigiPeçonha é um projeto acadêmico desenvolvido no curso de Ciência da Computação da UniMAX. O objetivo é digitalizar o registro de ocorrências de animais peçonhentos em Indaiatuba, conectando os cidadãos diretamente ao Centro de Controle de Zoonoses.

# 🤝 Trabalho em Equipe e Colaboração
The primary goal is to digitize the reporting process for venomous animals (scorpions, snakes, spiders) to provide faster triage and better data for public health authorities.

    * Risk Mapping: Identifying endemic areas and risk zones through geolocation.

    * Emergency Assistance: Providing immediate safety instructions and guidance during encounters.

    * Public Education: Reducing accidents through informative content on prevention.

    * Zoonosis Integration: Sending photos and metadata directly to specialists for analysis.

# 👩‍💻 My Role: Frontend Developer
In this project, I am responsible for architecting and implementing the User Interface (UI) and User Experience (UX) using React.js. My focus is on creating a responsive, accessible, and intuitive interface for all citizens.

Key Responsibilities & Features Implemented:
    * Authentication System: Developed the User Login and Registration flows.

    * User Profile Management: Built the Account/Profile section to track user data (age/weight) for risk group classification.

    * Dashboard & Navigation: Implemented the Main Menu and navigation structure.

    * Occurrence History: Created the visualization for users to track their past reports.

# 🚀 Technical Stack
    * Frontend: React.js (Vite)

    * Styling: CSS3 (Flexbox/Custom Properties)

    * State Management: React Hooks (useState, useEffect)

    * Design Tool: Excalidraw (UI/UX Prototyping)
